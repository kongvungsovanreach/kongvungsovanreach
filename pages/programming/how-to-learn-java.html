ជម្រាបសួរ​ លោក​ លោកស្រី​ ជា​មាតាបិតា​ ​អ្នកអាណាព្យាបាល​ និង​សាធារណជន​ទាំងអស់​! សួ​ស្តី​ប្អូន​ៗ សិស្សានុសិស្ស​ ខ្ញុំបាទ​ឈ្មោះ​ ជាង​ ប៊ុ​នលី​ ជានា​យក​ចាងហ្វាង​នៃ​សាលារៀន​ អន្តរទ្វីប​ អាមេរិកាំង​ សាខា​ម៉ៅសេទុង​ ខ្ញុំ​បាន​ទទួល​សញ្ញាបត្រ​គរុកោសល្យ​ក្នុង​ឆ្នាំ២០០៥​ បរិញ្ញាបត្រ​ ឆ្នាំ២០០៨​ និង​ទទួល​បាន​បរិញ្ញា​ជាន់ខ្ពស់​ ឆ្នាំ២០១៧​ និង​ចូល​បំរើ​ការងារ​ក្នុង​វិស័យ​អប់រំ​ចាប់តាំងពី​ឆ្នាំ២០០៥​ និង​បាន​ចូល​បំរើ​ការងារ​ជា​គ្រូបង្រៀន​ ពាក់​ពេញ​ម៉ោង​នៅ​សាលារៀន​ អន្តរទ្វីប​ អាមេរិកាំង​ ចាប់ពី​ឆ្នាំ២០០៩​។ សាលារៀន​ អន្តរទ្វីប​ អាមេរិកាំង​ បាន​យកចិត្ត​ទុកដាក់​បំពេញ​បេ​សកម្ម​អប់រំ​មួយ​ឈរ​លើ​មាគ៌ា​ដ៏​ត្រឹមត្រូវ​ ស្របតាម​កម្មវិធី​សិក្សា​របស់ក្រ​សួង​ អប់រំ​ យុវជន​ និងកីឡា​ សាលា​លើកកម្ពស់​ និង​លើកទឹកចិត្ត​សិស្ស​គ្រប់​កម្រិត​ថ្នាក់​ក្នុង​ការស្រាវជ្រាវ​តាម​ប្រព័ន្ធ​អ៊ី​នធឺ​ណេត​ និង​តាម​ឯកសារ​ក៏​ច្រើន​នៅក្នុង​បណ្ណាល័យ​ សាលា​មិន​ត្រឹមតែ​បណ្តុះបណ្តាល​សិស្សានុសិស្ស​ឱ្យ​ មានចំណេះ​ដឹង​ពិតប្រាកដ​ ក្រមសីលធម៌​ ថែមទាំង​ឱ្យ​ប្អូន​ចេះ​ចែករំលែក​ មាន​ទំនួល​ខុសត្រូវ​ក្នុង​សង្គម​ និង​មាន​សមត្ថភាព​គ្រប់គ្រាន់​នៅពេល​ដែល​ប្អូន​ចេញ​រៀន​នៅ​តាម​សកលវិទ្យាល័យ​នានា​មិនថា​ក្នុង​ប្រទេស​ ឬក្រៅប្រទេស​ទេ​ ហើយ​អនាគត​នឹង​ក្លាយជា​ធនធានមនុស្ស​ពោរពេញ​ដោយ​សមត្ថភាព​ទេពកោសល្យ​ សីលធម៌​ និង​មាន​ទំនួល​ខុសត្រូវ​ខ្ពស់​ក្នុង​សង្គម​។ សាលារៀន​ អន្តរទ្វីប​ អាមេរិកាំង​ គឺជា​ថ្នាល​បណ្តុះបណ្តាល​ដែល​មាន​ចក្ខុវិស័យ​ច្បាស់លាស់​គឺ​ “គុណភាព​ វិន័យ​ សេវាកម្ម​” សម្រាប់​សិស្សានុសិស្ស​មក​ពី​គ្រប់​មជ្ឈដ្ឋាន​ដើម្បី​ទទួលការ​អប់រំ​ប្រកបដោយ​ គុណភាព​ និង​ប្រសិទ្ធភាព​។ ជាចុងក្រោយ​សូម​ថ្លែងអំណរគុណ​មាតាបិតា​ និង​អ្នក​ អាណា​ ព្យាបាល​សិស្ស​ទាំងអស់​ ដែល​បាន​គាំទ្រ​ និង​មានទំនុក​ចិត្ត​ចំពោះ​សាលារៀន​ អន្តរទ្វីប​ អាមេរិកាំង​ ក៏ដូចជា​ផ្តល់​ឱកាស​ដល់​កូន​ៗឱ្យមក​សិក្សា​នៅ​សាលារៀន​ អន្តរទ្វីប​ អាមេរិកាំង​។ សូម​គោរព​ជូនពរ​ លោក​ លោកស្រី​ មាតាបិតា​ និង​អ្នកអាណាព្យាបាល​ និង​ជូន​ពរ​ប្អូន​ៗសិស្សានុសិស្ស​ទាំងអស់​មាន​សុខភាព​ល្អ​ សុភមង្គល​ និង​ជោគជ័យ​គ្រប់​ភារកិច្ច​។ ការ​វិនិយោគ​លើ​ចំណេះដឹង​កូនលោក​ អ្នក​ នាំមក​នូវ​ផលប្រ​យោ​ជន៉​ដ៏​ល្អ​បំផុត​។
